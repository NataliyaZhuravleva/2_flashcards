• JavaScript(JS)

    JS is a language.

  • Operator

    An operator is a special character (or characters) that indicates an action to be performed.

  • Variables

    A variable is a container that stores some information such as a number, an object, or a string.

  • Variable naming conventions

    Variable naming conventions is a set of rules for choosing the character sequence to be used for identifiers which denote variables, types, functions, and other entities in source code and documentation.

  • Functions

    Function is a block of code that performs an action and returns a result; optionally takes arguments.

  • Methods

    A Method is an action run on a piece of data.

  • Arguments

    Argument is what we pass into a function or method.

  • Parameters

    Parametr is a variable that is assigned to the argument.

  • Return

    The return statement ends function execution and specifies a value to be returned to the function caller.

  • Chaining Methods

    Methods that are chained to occur after another method

  • Strings

    Strings represent a line of text

  • Booleans

    Booleans are yes/no or true/false statements

  • Undefined

    Undefined is a value for variables which have not been assigned a variable yet

  • NaN

    Not a Number, but is considered a number

  • Null

    Null means null

  • Alerts

    Alerts are pop-ups to offer information to the website's visitor

  • Comments

    Comments are developer notes left in code which are not read to be real code

  • JQuery

    JQuery is a library.

  • Attributes

    Attributes are additional information provided to HTML tags (ie href)
